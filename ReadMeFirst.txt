1.Prepare you system
	Copy .nshrc to the home folder of you system.
	This will allow for easy setup upon execution of the program.

2.Build the project
	Using a system with a gcc compiler installed go to the project
	directory in command-line and type "make". This will compile the
	project into a program that you can run.

3.Execute the program
	After the above steps now you are ready to run the program! Type
	"./nsh6” into the command-line and the program should start.  For
	a list of commands and some more details about the program just
	type "help" into the command-line.
